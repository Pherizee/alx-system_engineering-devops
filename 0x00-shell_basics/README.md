# Shell Basics Scripts

This directory contains several scripts in line with shell basics project

## 0-current_working_directory

This script prints the path name of the current working directory

## 1-listit

This script lists all files and directories in the current working directory

## 2-bring_me_home

This script changes the working directory to the user's home directory

## 3-listfiles

This script lists contents of the current working directory in the long format

## 4-listmorefiles

This will list contents of the current working directory including hidden files in the long format

## 5-listfilesdigitonly

This script lists all contents for the working directory in the:
* Long format
* with user and group IDs as numerical values
* and hidden files

## 6-myfirstdirectory

This script creates a directory named my_first_directory in the /tmp/ directory

## [7-movethatfile](./7-movethatfile)

This script moves the file betty from /tmp/ to /tmp/my_first_directory

## [8-firstdelete](./8-firstdelete)

This script deletes the file betty from /tmp/my_first_directory

## [9-firstdirdeletion](./9-firtstdeletion)

This script deletes the directory /tmp/my_first_directory

## [10-back](./10-back) 

This script return the user back to his previous directory. Working like a back button on GUI.

## [11-lists](./11-lists)

This script lists all files (including hiddent one)s in the current directory, parent and /boot directory

## [12-file_type](./12-file_type)

This script print the type of a file /tmp/iamafile

## [13-symbolic_link](./13-symbolic_link)

This script creates a symbolic link to the /bin/ls named __ls__ in the working directory

## [14-copy_html](./14-copy_html)

This script copies all the HTML files from the current directory to the parent directory without duplicating

## [100-lets_move](./100-lets_move)

This moves all files beginning with an uppercase letter from the working directory to /tmp/u

## [101-clean_emacs](./101-clean_emacs)

This script deletes all files in the working directory that end with the tilde (~) character

## [102-tree](./102-tree)

This script creates the directories welcome/, welcome/to/ and welcome/to/school/ in the working directory without repetition of command.

## [103-commas](./103-commas)

This script lists lists all files in a working directory:

* separated by commas (,),
* directory names ending with a slash (/), and
* hidden files and directories`
